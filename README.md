# Personal Site

My personal website built with Astro, featuring a blog and case studies.

## Tech Stack

- Astro
- TypeScript
- MDX for content
- Procedural SVG image generation

## Getting Started

```bash
pnpm install
pnpm dev
```

Visit `http://localhost:4321`

## Build

```bash
pnpm build
```

## Project Structure

```
scripts/          # Image generation utilities
src/
  components/     # Reusable UI components
  content/        # Blog posts and case studies (MDX)
  layouts/        # Page layouts
  pages/          # Routes and pages
public/           # Static assets
```
